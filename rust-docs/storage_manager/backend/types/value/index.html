Runtime typed values and their binary encoding/decoding.

DataValue is the in-memory representation of any SQL column value. The two primary operations are:

  • DataValue::parse_and_encode — parse a raw string literal and return the binary encoding ready for on-disk storage.
  • DataValue::from_bytes — decode the binary form back into a typed value.
