storage_manager::backend::types

Expand description

SQL-99 comparison semantics for typed values.

Provides:

  • ComparisonError — the typed error returned for incompatible types.
  • Comparable — the comparison trait implemented by DataValue.
  • compare_nullable / nullable_equals — NULL-aware wrappers that return None (UNKNOWN) when either operand is NULL, matching SQL semantics.
